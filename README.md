# 科学范式转变与AI时代的认识论革命

一个交互式教育网站，基于托马斯·库恩的《科学革命的结构》，展示从自然哲学到AI时代的六次科学范式转变。

## 功能特色

- **完整的科学史叙述**：涵盖从亚里士多德到AI时代的六次范式转变
- **双语支持**：中英文一键切换
- **文本转语音**：每个章节支持TTS朗读
- **AI智能助手**：集成DeepSeek API进行科学对话
- **响应式设计**：适配各种设备屏幕
- **现代交互**：使用Vue.js 3构建
